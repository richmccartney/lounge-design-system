# Button

`Solid orange buttons` = primary page level actions

`Outline orange buttons` = secondary page level actions

Only one primary page level button (solid) can appear per page, because they are
meant to direct users into taking the action you want them to take. They should
highlight the most important, forward-moving action on the page. Examples of
forward-moving actions are `Next`, `Submit`, `Continue`.

The primary page level button can be used by itself or in combination with the
secondary page level button (outline). The secondary page level button should
never be used alone/without the primary page level button.

When using both primary and secondary buttons together, place both buttons to
the bottom right corner of the page with 24px padding from the right and bottom.
The secondary button should be placed to the left of the primary button with
24px in between.
