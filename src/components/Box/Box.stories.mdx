# Box

The Box component is a low-level primitive used for layout purposes. It can be
used to create grid layouts, apply padding or margin, and more.

Here is an example of how the Box component can be used to apply padding and
border around its children.

<Story name="Box">
  <Box p={4} border="1px" borderColor="blue.500" height="100%">
    This is a box.
  </Box>
</Story>
